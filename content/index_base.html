Data Levers

Ecosystems benefit from some balance of growth and fire. Sometimes we dig new channels for water to flow and sometimes we dam a river. "Data levers" allow us to reconfigure how the data records we generate flow to AI systems.

You are a crucial source of records needed to train and test AI. If you support an organization or an application of AI, you may want your data contributions flow to the corresponding systems. If you dislike how an organization is using AI, you may want to stop the data flow.

The purpose of this page is link to tools -- software you can download, browser extensions you can install, and online platforms you can visit -- that help "pull" data levers. For each link, we provide brief commentary on: costs to use them, incentives, and current targets. To have impact, data levers require collective participation. By curating a list of tools, we can collective action more feasible. To learn more or to contribute, visit About+Contact. If you're interested in measuring the impact of data levers, see The Impact of Levers.
