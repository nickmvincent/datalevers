Measuring Impact

Measuring the impact of data levers is a scientific process.

More precisely, we can pose a number of questions along the lines of: Given some set of people (we'll call the set S) take some action to add, withhold or alter some data records they generate (we can call the dataset D), how will affected AI systems change in terms of their capabilities?

For instance, how will AI art capabilities change if a firm releases a model with no externally scraped data? Or, how will code-focused language model capabilities change after offering GitHub users an opt-out opportunity? For now, this page just includes a list of systems to watch. In the future, we'll directly compare different capabilities of lever-impacted systems.
